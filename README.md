# Analizador de Gastos Web

Aplicación web desarrollada con Flask y Pandas para analizar gastos a partir de archivos CSV.

## Características

- Carga de archivos CSV
- Cálculo del gasto total
- Agrupación de gastos por categoría
- Verificación de presupuesto
- Interfaz web con Bootstrap
- Tabla de resultados

## Tecnologías utilizadas

- Python
- Flask
- Pandas
- Bootstrap 5
- HTML
- CSS

## Instalación

1. Clonar el repositorio

```bash
git clone https://github.com/juanexzedh/analizador-gastos-web.git
```

2. Instalar dependencias

```bash
pip install -r requirements.txt
```

3. Ejecutar la aplicación

```bash
python app.py
```

## Formato esperado del CSV

```csv
fecha,categoria,monto
2025-01-01,Comida,25000
2025-01-02,Transporte,10000
2025-01-03,Ocio,15000
```
